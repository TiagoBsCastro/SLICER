\doxysubsection{Detailed Description}
Class to model the universe using the \mbox{\hyperlink{classw0waCDM}{w0wa\+CDM}} cosmological model. 

This class provides methods to compute cosmological distances such as comoving distance and transverse comoving distance under the \mbox{\hyperlink{classw0waCDM}{w0wa\+CDM}} model, which includes parameters for the equation of state of dark energy that vary with redshift.
